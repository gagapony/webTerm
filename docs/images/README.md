# Screenshots

Drop the screenshots referenced by the top-level [`README.md`](../../README.md) here.
Each file in the README is collapsible (`<details>`); replacing a path below makes it
appear when a reader expands that section.

Suggested dimensions: **1920×1080** PNG, captured at the browser's full width.
Optimize with something like `oxipng` or `pngquant` to keep the repo small.

| File | What to capture |
|------|-----------------|
| `main-interface.png` | The primary workspace: frosted topbar, session tabs, terminal over a background |
| `multi-session.png` | Several SSH/Telnet sessions open at once across tabs |
| `connection-manager.png` | The saved-connections dropdown with a few profiles |
| `recording.png` | A recording indicator on a tab + a `.cast` being replayed |
| `settings-themes.png` | Settings panel → theme picker (Default + Catppuccin Mocha) & transparency |
| `backgrounds.png` | Terminal over Forest / Mountain / Ocean presets with blur & opacity tuned |
| `osc52-clipboard.png` | Copying yanked text from remote `nvim`/`tmux` into the browser clipboard |
| `login.png` | The login card over a frosted background |
| `hero.png` *(optional)* | A wide hero banner for the top of the README |
