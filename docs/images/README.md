# Screenshots

Drop the screenshots referenced by the top-level [`README.md`](../../README.md) here.
The `main-interface*` shots render as a 2×2 grid at the top of the Screenshots
section; the rest appear inside collapsible (`<details>`) blocks.

Suggested dimensions: **1920×1080** PNG, captured at the browser's full width.
Optimize with something like `oxipng` or `pngquant` to keep the repo small.
